### Kurulum

1. Depoyu klonlayın:

```bash
git clone https://github.com/Fuywkin/time-tracker.git
```

2. Proje dizinine gidin:

```bash
cd time-tracker
```

3. Gerekli paketleri yükleyin:

```bash
npm install
```

4. Uygulamayı başlatın:

```bash
npm run dev
```

5. Tarayıcınızdan `http://localhost:3000` adresine gidin.
